## 0 inbox

- **[remote procedure call](https://en.wikipedia.org/wiki/Remote_procedure_call)** (**RPG**): when a computer program causes a procedure to execute in a different address space, which is coded as if it were a local procedure call, without the programmer explicitly coding the details for the remote interaction.
- [**interface definition language**](https://en.wikipedia.org/wiki/Interface_description_language) or **interface description language** (**IDL**): is a generic term for a language that lets a program or object written in one language communicate with another program written in an unknown language.
